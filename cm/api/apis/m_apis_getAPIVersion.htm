GET /api/apis/versions/{APIVersionID}

Returns the specified API version.

Authorization Roles/Permissions: Anyone can use this operation; login is not required.

HTTP Method

GET

Back to top

URL

api.acmecorp.com/api/apis/versions/{ApiVersionID}[?includeTargetAPI=boolean][&includeOperations=boolean]

Back to top

Sample Request

The example below shows a request for the API version object for apiversion18971.acmecorp.

Sample request URL #1

https://<hostname>/api/apis/versions/apiversion18971.acmecorp

Sample request URL #2

https://<hostname>/api/apis/versions/apiversion11793.acmecorp?includeTargetAPI=true&includeOperations=true

Sample request headers

GET /api/apis/versions/apiversion18971.acmecorp HTTP/1.1
Host: <hostname>
Accept: application/json, text/javascript, */*; q=0.01

Sample request body

Not applicable.

Request Parameters

Parameter Parm Type Data Type Required Description
APIVersionID Path string Required The unique ID for the specified API version.
includeOperations Query boolean Optional If set to true, operations associated with the API version are returned. Default: false. See example.
includeTargetAPI Query boolean Optional If set to true, Target API, if applicable, is returned. Default: false.

Back to top

Response

If successful, this method returns HTTP status code 200, with information about the requested API version.
