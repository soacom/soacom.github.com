PUT /api/apis/versions/{APIVersionID}/oauthdetails

Adds or modifies the OAuth details for the specified API version.

Authorization Roles/Permissions: To complete this operation successfully, a user must be logged in and must have modify permission for the API; an API Admin or Business Admin.

Sample Request

The example below saves both OAuth 1.0a and OAuth 2.0 settings.

Sample request URL

https://<hostname>/api/apis/versions/apiversion18973.acmecorp/oauthdetails

Sample request headers

PUT /api/apis/versions/apiversion18973.acmecorp/oauthdetails HTTP/1.1
Host: <hostname>
Accept: application/json, text/javascript, */*; q=0.01
Content-Type: application/json; charset=UTF-8

Sample request body #1

{
  "APIVersionID":"apiversion18973.acmecorp",
  "ProxyAPI":{
    "ProductionEndpoint":{
      "OAuthProvider":"com.soa.oauth.provider.third-party",
      "OAuth20Supported":true,
      "OAuth10aSupported":true,
      "OAuth20Settings":{
        "AuthorizationEndpointURL":"http://acmeprovider.authserver20.com",
        "TokenEndpointURL":"http://acmeprovider.tokenserver20.com",
        "TokenEndpointRequestVerb":"POST",
        "BearerTokenMethods":{
          "BearerTokenMethod":[
            "com.soa.oauth.bearer.token.method.form.param"
          ]
        }
      },
      "SupportedGrant":[
        "password"
      ],
      "ClientAuthPolicyOption":[
        "com.soa.oauth.client.security.basicauth"
      ],
      "OAuth10aSettings":{
        "AuthorizationEndpointURL":"https://acmeprovider.authserver10a.com",
        "AccessTokenEndpointURL":"https://acmeprovider.accesstoken10a.com",
        "RequestTokenEndpointURL":"https://acmeprovider.requesttoken.com",
        "TokenEndpointRequestVerb":"GET",
        "TokenParameterTransmissionMethods":{
          "TokenParameterTransmissionMethod":[
            "com.soa.oauth.token.parameter.transmission.method.header"
          ]
        }
      }
    },
    "SandboxEndpoint":{
      "OAuthProvider":"OAuth Provider1",
      "OAuth20Supported":true,
      "OAuth10aSupported":false
    }
  }
}

Sample request body #2

{
  "APIVersionID":"apiversion18972.acmecorp",

}
