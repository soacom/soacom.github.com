DELETE /api/apis/{APIID}

Deletes the specified API and a single version, which must be the only version. If there are additional API versions, they must be deleted first, using the DELETE /versions/{APIVersionID} operation.

Authorization Roles/Permissions: To complete this operation successfully, a user must be logged in and must be an Administrator for the API.
