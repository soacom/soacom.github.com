GET /api/apis/{APIID} Method

Returns information about an API.

Authorization Roles/Permissions: Anyone can use this operation; login is not required However, the results that are returned are restricted based on the user's access rights. For example, content relating to a Private API is not returned unless the user is logged in and has access rights to the API. If at least one version of a Private API is public, the public content is returned. If all versions of the API are private, this operation will only return the content for invited users, API admins, business admins, and site admins.

Request Parameters

Parameter Parm Type Data Type Required Description
APIID Path string Required The unique ID for the specified API.
includeDefaultVersion Query boolean Optional Indicates whether to include the default version in the information returned. If not specified, defaults to false.

Response

If successful, this method returns HTTP status code 200, with information about the API.
