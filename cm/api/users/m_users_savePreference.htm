POST /api/users/{UserID}/preferences

Changes the value of a user preference.

Authorization Roles/Permissions: Authorized User or Site Admin.

HTTP Method

POST

Back to top

URL

https://<hostname>/api/users/{UserID}/preferences

Back to top

Sample Request

The example below shows a change in user preference regarding display of the Dashboard help.

Sample request URL

https://<hostname>/api/users/user10002.acmecorp/preferences/

Sample request headers

POST /api/users/user10002.acmecorp/preferences/ HTTP/1.1
Host:<hostname>Accept: */*
Content-Type: application/json; charset=UTF-8

Sample request body

{
  "Name":"com.soa.helptext.dashboard.helpClosed",
  "Value":"open"
}
