PUT /api/users/{UserID}/password Method

Modifies the password of a user in the system.

Authorization Roles/Permissions: Authorized User only.

HTTP Method

PUT

Back to top

URL

https://<hostname>/api/users/{UserID}/password

Back to top

Sample Request

In the example below, the user changes the password from Margaret23 to Margaret14.

Request URL

https://<hostname>/api/users/user10011.open/password

Sample request headers

PUT /api/users/user10011.open/password HTTP/1.1
Host: <hostname>
Accept: text/plain, */*; q=0.01
Content-Type: application/json; charset=UTF-8

Sample request body

{
  "OldPassword":"Margaret23",
  "NewPassword":"Margaret14"
  "NewEmail":"jane.mead@example.com"
}

Request Parameters

Parameter Parm Type Data Type Required Description
UserID Path string Required The unique ID for a specific user.
PasswordChange Body PasswordChange Required Contains information about a change of user password or email address.

Response

If successful, this method returns HTTP status code 200, with the UserID in plain text.
