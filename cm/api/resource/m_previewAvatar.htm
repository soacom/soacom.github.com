GET /api/users/{UserID}/previewavatar Operation

Returns a preview of a specific user's avatar from the specified avatar provider.

Note: This operation is currently applicable only to users. Preview is not available for other resources that have avatars, such as apps, APIs, and groups.

Authorization Roles/Permissions: Anyone can use this operation; login is not required.

URL

Sample Request

Request URL #1

Image is uploaded to the platform.

https://<hostname>/api/users/user12262.open/previewavatar?Provider=com.soa.imageprovider.type.default

Request URL #2

Image is supplied by Gravatar.

https://<hostname>/api/users/user12262.open/previewavatar?Provider=com.soa.imageprovider.type.gravatar

Sample request headers

GET /api/users/user13998.open/previewavatar?Provider=com.soa.imageprovider.type.default HTTP/1.1
Host: <hostname>
Accept: image/png,image/*;q=0.8,*/*;q=0.5

Request Parameters

Parameter Parm Type Data Type Required Description
UserID Path String Required The unique UserID for the user for which the avatar is being previewed.
Provider Query String Required The avatar provider; for example, com.soa.imageprovider.type.default. For a list of valid values, see Avatar Providers.

Response

If successful, this method returns HTTP status code 200, with the avatar image in the response.
