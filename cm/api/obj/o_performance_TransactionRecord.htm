TransactionRecord Object

Contains information about a transaction for an app or API.

This object is used by the following operations:

  • GET /api/apis/versions/{APIVersionID}/txlogs/{TransactionID}
  • GET /api/apps/versions/{AppVersionID}/txlogs/{TransactionID}

The properties included in the TransactionRecord object are listed below.

Property Type Description
ClientHost string The IP address for the client host.
APIVersionName string The name of the API version.
MethodKey string The name of the operation invoked in the transaction.
AppVersionName string The name of the app version.
AppRuntimeID string Unique ID used at runtime by the app version.
ResponseTime Duration Response time for the app.
RequestMessageSize unsignedLong Request message size in bytes.
ResponseMessageSize unsignedLong Response message size in bytes.
ResponseErrorMessage string Response error message, if any.
APIResponseTime Duration Response time for the API.
APIURL string URL for the API.
RequestMessage RequestMessage Information about the request message.
ResponseMessage ResponseMessage Information about the response message.
FaultMessage FaultMessage Information about the fault message if any.
TransportHeaders string Information about the HTTP headers.
