GoverningWorkflowChangeRequest Object

Contains information about the workflow governing a resource.

This object is used by the following operations:

  • GET /api/apps/versions/{RuntimeID}/governingworkflow
  • PUT /api/apps/versions/{AppVersionID}/governingworkflow

The properties included in the GoverningWorkflowChangeRequest object are listed below.

Property Type Description
Governed boolean Indicates whether the resource is governed by a custom workflow (true) or not (false).
Key Key The unique ID for the custom workflow.
Name string The name of the custom workflow.
Description Key The description of the custom workflow.
State Key The current state of the workflow.
Updated dateTime The date/time stamp when the resource was last updated. Used only in response messages.
