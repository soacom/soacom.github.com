Discussion Object

Includes information about a discussion.

This object is used by the following operations:

  • POST /api/discussions
  • GET /api/discussions/{DiscussionID}

The parameters included in the discussion object are listed below.

Property Type Required Description
DiscussionID string Optional The unique ID for a specific discussion.
DiscussionSubjectID string Optional The unique ID for the specific resource that a discussion relates to; for example, the AppVersionID, APIVersionID, GroupID, or a custom BoardID.
Discussion Type string Optional The type of discussion. Deprecated. Might be returned in a response object.
Topic string Optional The topic of the discussion, as specified by the user starting the discussion.
Content string Optional The content of the discussion.
UserID string Required The unique ID for a specific user..
UserName string Optional The username of the author of the discussion.
Created dateTime Optional The date/time stamp when the resource was first created. Used only in response messages.
Comments int Optional The number of comments on the discussion.
Marks int Optional The number of marks on the discussion.
IsMarked boolean Optional Indicates whether there are any marks on the discussion or not.
Tag string[ ] Optional One or more user-defined tags assigned to the discussion.

Visibility string Optional The visibility setting for this item; either Public, which means everyone can see it, or Limited, which means it can only be seen by team members. For valid values, see Visibility Settings
pubDate dateTime Optional The publication date for the discussion, formatted in Coordinated Universal Time (UTC).
