PUT /api/apps/versions/{RuntimeID}/oauthclient Method

Changes one or more OAuth client properties

Authorization Roles/Permissions: App team member, Business Administrator

HTTP Method

PUT

URL

https://<hostname>/api/apps/versions/{RuntimeID}/oauthclient

Sample Request

The example below shows an update request for the OAuth client properties.

Sample request URL

https://<hostname>/api/apps/versions/9CA3Avaz8e96zMUKhaL4mM4s.acmecorp/oauthclient

Sample request headers

PUT /api/apps/versions/9CA3Avaz8e96zMUKhaL4mM4s.acmecorp/oauthclient HTTP/1.1
Host: <hostname>
Accept: */*
Content-Type: application/json; charset=UTF-8

Sample request body

{
  "RedirectURI":"http://www.example.com",
  "OAuthClientType":"com.soa.oauth.clienttype.public"
}

Request Parameters

Parameter Parm Type Data Type Required Description
RuntimeID Path string Required Unique ID used at runtime by the app version.
OAuthClient Body OAuthClient Required Includes information about the OAuth Client settings for the specified app. The RedirectURI and OAuthClientType are required in the request.
