DELETE apps/{AppID} Method

Deletes an app (all versions).

Note: If you want to completely remove an app, use DELETE /api/apps/{AppID}. If you want to delete the only existing app version, but keep the app, first add a new version, using POST apps/{AppID}/versions, and then delete the old version using DELETE /api/apps/versions/{RuntimeID}. An app cannot exist in the Community Platform without at least one valid version.

Authorization Roles/Permissions: App team member, Business Administrator
