PUT apps/versions/{AppVersionID}/secret Method

Regenerates the shared secret value for the specified app.

For more information about the shared secret value, see Authenticating with the Shared Secret.

Note: This operation uses the updatePassword method which is part of Policy Manager. The Community Platform generates the SharedSecret value and passes the value to the Policy Manager updatePassword method so that it can be stored in an encrypted form.

Authorization Roles/Permissions: App team member, Business Administrator
