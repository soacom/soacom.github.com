Legals Service: Overview

The Legals service provides the ability to manage legal agreements in the Community Platform.

URL: https://<hostname>/api/legals

Action Endpoint Description
accept agreement POST /api/legals/agreements Records the user's acceptance of a legal agreement document.
create new agreement POST /api/legals Adds information about a new legal agreement for the specified API version.
get agreement document GET /api/legals/{DocumentID} Returns a specified legal agreement document associated with an API.
get agreement documents GET /api/legals Returns one or more legal agreement documents associated with a resource such as an API.
