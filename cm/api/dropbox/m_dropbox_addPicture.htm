POST /api/dropbox/pictures Method

Adds an image for a resource such as a user profile. Provides a temporary storage location for an avatar image the user uploads as part of creating or updating a resource such as a user profile, an API, or an app.

For more information on managing images associated with resources on the platform, refer to Managing Avatars on the Platform.

Authorization Roles/Permissions: To complete this operation successfully, a user must be logged in and must be either an authorized user for the resource or a site admin.

Response Body

Name Type Description
DropboxID string The unique ID for a picture on the platform, such as the avatar for an app, API, or user. The picture ID is assigned by the Dropbox when the picture is first uploaded, using the Dropbox service, and stored in a temporary Dropbox location as part of adding or updating the resource. It is then sent to the server where it is permanently stored.
